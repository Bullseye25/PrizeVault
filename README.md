## ğŸ›  Technology Stack (On-Chain Backend)
Our "Backend" is fully decentralized, running on the **Base Blockchain**.

*   **Smart Contracts**: Solidity (Logic, State, Deposits).
*   **Randomness**: Chainlink VRF (Verifiable Random Function) for tamper-proof draws.
*   **Frontend**: Next.js 14 + Tailwind CSS (Glassmorphism UI).
*   **Wallet**: RainbowKit + Wagmi.

### Smart Contract Logic (`/contracts/PrizeVault.sol`)
*   **`deposit(amount)`**: Securely locks user funds and issues 'tickets'.
*   **`drawWinner()`**: Calculates the prize (Yield) and pseudo-randomly selects a winner.
*   **`withdraw()`**: allowing users to reclaim their principal 1:1.
